# BOOK PREFERENCES

- TONE: Optimistic, Athlete Centric, Coach-Sceptic, small irony on old outdated methods, Hands-On
- INCLUSIVENESS: use gender-neutral form where it's easily readable; if not: switch between male and female form; sometimes include examples for persons with disabilities
- ADDRESS: use german "Sie" when addressing parents and old-school coaches; use german "Du" when addressing athletes and new-model coaches
